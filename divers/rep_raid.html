L'utilisation des procédés RAID1 et RAID5
Réponse du test

Les bonnes réponses aux questions apparaissent en couleur rouge.
Question 1 :
Quel est le type de disque système qui réalise une copie exacte de toutes les données d'une partition disque dans une autre partition ?

  • RAID.
  • Sauvegarde disque.
  • Mise en miroir.
  • Dépannage à chaud.

    Question 2 :
    Quel est le type de disque système qui utilise de 3 a 32 disques physiques et est aussi connu sous le nom de RAID5 ?

  • Agrégats par bandes avec parité.
  • Agrégats par bandes sans parité.
  • Disques en miroir.
  • Agrégats par bandes avec données répétitives.

    Question 3 :
    Quel sont les types de disques qui assurent la tolérance de pannes ?

  • Les agrégats de partition.
  • Les agrégats par bandes avec parité.
  • Les agrégats par bandes sans parité.
  • Les disques en miroir.
  • RAID de niveau 0.
